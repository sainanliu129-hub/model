# E1 机器人 PD 控制仿真说明

> 动力学计算说明见同目录 [E1_DYNAMICS_README.md](E1_DYNAMICS_README.md)。本文仅说明 PD 控制仿真脚本用法。

## 概述

`sim_e1_pd_control.m` 实现了完整的 PD 控制流程，适用于**悬空/吊装情况**（无接触力和接触摩擦力）。

## 功能特点

✅ **完整的控制流程**：
- 期望轨迹生成（正弦波）
- PD控制器计算力矩
- 动力学仿真（考虑关节摩擦力）
- 状态更新循环

✅ **无接触力**：
- 不考虑接触力（机器人悬空）
- 不考虑接触摩擦力
- 只考虑关节摩擦力（可选）

✅ **灵活配置**：
- 可配置控制参数（Kp, Kd）
- 可配置摩擦力参数
- 可选择活动关节

## 使用方法

### 基本使用

```matlab
% 直接运行脚本
sim_e1_pd_control
```

### 修改参数

在脚本中修改以下参数：

```matlab
% PD控制参数
control_params.Kp = 30;         % 位置增益
control_params.Kd = 0.5;        % 速度增益

% 关节摩擦力参数
friction_params.enable = true;  % 是否启用摩擦力
friction_params.tau_c = 0.5;    % 库伦摩擦
friction_params.b = 0.1;        % 粘滞摩擦
friction_params.tau_s = 0.8;     % 静摩擦

% 选择活动关节
active_joints = 1:12;  % 只控制腿部关节
```

## 控制流程

```
┌─────────────────────────────────────────────────────────┐
│ 控制循环（500Hz，周期0.002秒）                          │
├─────────────────────────────────────────────────────────┤
│ 1. 读取当前状态                                        │
│    q, qd (关节位置、速度)                              │
├─────────────────────────────────────────────────────────┤
│ 2. 计算控制误差                                        │
│    e_pos = q_desired - q                                │
│    e_vel = qd_desired - qd                              │
├─────────────────────────────────────────────────────────┤
│ 3. PD控制器计算力矩                                     │
│    τ_control = τ_ff + Kp×e_pos + Kd×e_vel             │
├─────────────────────────────────────────────────────────┤
│ 4. 计算关节摩擦力（可选）                               │
│    τ_friction = f(qd)                                  │
├─────────────────────────────────────────────────────────┤
│ 5. 计算总力矩                                          │
│    τ_total = τ_control - τ_friction                    │
│    （不考虑接触力和接触摩擦力）                         │
├─────────────────────────────────────────────────────────┤
│ 6. 动力学求解（正动力学）                               │
│    M(q)*qdd + C(q,qd) + G(q) = τ_total                │
│    qdd = M^(-1) * (τ_total - C - G)                   │
├─────────────────────────────────────────────────────────┤
│ 7. 状态更新（数值积分）                                 │
│    qd_new = qd + qdd * dt                              │
│    q_new = q + qd * dt + 0.5 * qdd * dt^2             │
└─────────────────────────────────────────────────────────┘
```

## 关键公式

### PD控制器

```
τ_control = τ_ff + Kp × (q_desired - q) + Kd × (qd_desired - qd)
```

### 关节摩擦力

```
运动时（|qd| ≥ ε）：
  τ_friction = τ_c × sign(qd) + b × qd

静止时（|qd| < ε）：
  |τ_friction| ≤ τ_s
```

### 动力学方程

```
M(q) × qdd + C(q,qd) + G(q) = τ_total

其中：
  τ_total = τ_control - τ_friction
```

### 状态更新

```
qd_new = qd + qdd × dt
q_new = q + qd × dt + 0.5 × qdd × dt^2
```

## 参数说明

### 仿真参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `dt` | 0.002 s | 控制周期（500Hz） |
| `t_end` | 5 s | 仿真时间 |

### PD控制参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `Kp` | 30 | 位置增益 (N·m/rad) |
| `Kd` | 0.5 | 速度增益 (N·m·s/rad) |
| `tau_ff` | 0 | 前馈力矩 (N·m) |

### 摩擦力参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `enable` | true | 是否启用摩擦力 |
| `tau_c` | 0.5 | 库伦摩擦系数 (N·m) |
| `b` | 0.1 | 粘滞摩擦系数 (N·m·s/rad) |
| `tau_s` | 0.8 | 静摩擦 (N·m) |
| `epsilon_vel` | 0.01 | 低速阈值 (rad/s) |

## 注意事项

1. **无接触力**：该仿真适用于悬空/吊装情况，不考虑接触力和接触摩擦力

2. **关节摩擦力**：可以启用或禁用关节摩擦力，通过 `friction_params.enable` 控制

3. **活动关节**：可以通过 `active_joints` 选择要控制的关节（例如只控制腿部）

4. **数值积分**：使用欧拉法进行数值积分，对于更高精度可以使用 Runge-Kutta 方法

5. **控制频率**：默认500Hz（0.002s），可以根据需要调整

```matlab

G = inverseDynamics(robot, q', zeros(size(qd')), zeros(size(qd')));
tau_ff = G';  % 重力补偿作为前馈
```

```matlab
q_desired(13:18, :) = 0;  % 手臂关节期望位置为0

```

## 故障排除

### 问题1：仿真不稳定

**原因**：Kp或Kd过大导致振荡

**解决**：减小Kp或增大Kd

### 问题2：跟踪误差大

**原因**：Kp过小或摩擦力过大

**解决**：增大Kp或减小摩擦力参数

### 问题3：计算速度慢

**原因**：控制频率过高或关节数过多

**解决**：增大dt或减少活动关节数
