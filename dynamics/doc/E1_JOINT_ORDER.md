# E1 机器人关节顺序说明

## 关节顺序（与 URDF 一致，全代码统一以此为准）

根据 E1.urdf 中关节出现顺序，**整机 18 维配置为：左臂 → 右臂 → 左腿 → 右腿**。  
所有整机 18 维向量（q、qd、qdd、τ）及动力学接口（如 `compute_e1_dynamics`、`compute_e1_forward_dynamics_full`）均按此顺序，不得使用“腿 1:12、臂 13:18”等其它约定。

### 关节 1-3：左臂

| 关节编号 | 关节名称 | 说明 |
|---------|---------|------|
| **1** | `arm_l1_joint` | 左臂 1 |
| **2** | `arm_l2_joint` | 左臂 2 |
| **3** | `arm_l3_joint` | 左臂 3 |

### 关节 4-6：右臂

| 关节编号 | 关节名称 | 说明 |
|---------|---------|------|
| **4** | `arm_r1_joint` | 右臂 1 |
| **5** | `arm_r2_joint` | 右臂 2 |
| **6** | `arm_r3_joint` | 右臂 3 |

### 关节 7-12：左腿

| 关节编号 | 关节名称 | 说明 |
|---------|---------|------|
| **7** | `leg_l1_joint` | 左腿 1 |
| **8** | `leg_l2_joint` | 左腿 2 |
| **9** | `leg_l3_joint` | 左腿 3 |
| **10** | `leg_l4_joint` | 左腿 4 |
| **11** | `leg_l5_joint` | 左腿 5 |
| **12** | `leg_l6_joint` | 左腿 6 |

### 关节 13-18：右腿

| 关节编号 | 关节名称 | 说明 |
|---------|---------|------|
| **13** | `leg_r1_joint` | 右腿 1 |
| **14** | `leg_r2_joint` | 右腿 2 |
| **15** | `leg_r3_joint` | 右腿 3 |
| **16** | `leg_r4_joint` | 右腿 4 |
| **17** | `leg_r5_joint` | 右腿 5 |
| **18** | `leg_r6_joint` | 右腿 6 |

## 关节分组

- **左臂**：1-3 (`arm_l1` 到 `arm_l3`)
- **右臂**：4-6 (`arm_r1` 到 `arm_r3`)
- **左腿**：7-12 (`leg_l1` 到 `leg_l6`)
- **右腿**：13-18 (`leg_r1` 到 `leg_r6`)

## 在代码中的使用

```matlab
% 整机 18 维：左臂 1-3, 右臂 4-6, 左腿 7-12, 右腿 13-18
q = zeros(18, 1);
q(1:3)   = ...;  % 左臂
q(4:6)   = ...;  % 右臂
q(7:12)  = ...;  % 左腿
q(13:18) = ...;  % 右腿

% 仅腿（12 维）
leg_idx = 7:18;   % 左腿 7-12 + 右腿 13-18
q_leg = q(leg_idx);
```
