# 两种激励轨迹方案对齐说明

为便于对比 **run_excitation_trajectory_standalone（fmincon）** 与 **TLBO_E1（TLBO）** 的效果，两种方案在以下方面已对齐。

## 1. 原始机器人文件

| 项目 | 说明 |
|------|------|
| 模型 | E1，同一 URDF：`noetix_description/urdf/E1.urdf` 或 `urdf/E1.urdf` |
| 肢体 | `get_e1_limb_robot(limb)`，limb = `'left_leg'` / `'right_leg'` / `'left_arm'` / `'right_arm'` |
| 回归矩阵 | `ReMatrix_E1_limb_URDF(limb, refPos, refVel, refAcc, 1, 1)` |
| 扭矩限值 | `max_effort`：腿从 URDF `get_limb_effort_from_urdf` 读取，臂或缺失时 50 N·m |

## 2. 约束（完全一致）

| 项目 | 数值/来源 |
|------|-----------|
| 关节位置 | 默认 `[−2π, 2π]`（每轴）；可选紧限位见 standalone 注释 |
| 关节速度 | `[16.75; 20.1; 20.1; 13.18; 12.46; 12.46]` rad/s（腿）；臂 `[10;10;10]` |
| 关节加速度 | `[200; 200; 200; 200; 200; 200]` rad/s²（腿）；臂 `[100;100;100]` |
| 周期 | 8 s |
| 采样 | 100 Hz（单周期 801 点） |

- **standalone**：已设 `config.use_full_bounds_for_optim = true`，不再使用 0.4 范围/0.9 速度收紧，优化与校核均用上述同一套限位。
- **TLBO_E1**：`q_max/q_min/dq_max/ddq_max` 与上表一致，`Tradeoff_Modify` 在该限位内缩放。

## 3. 校核（完全一致）

| 项目 | 说明 |
|------|------|
| 校核内容 | 位置、速度、加速度、扭矩（|τ| ≤ 0.9×max_effort） |
| 采样点数 | 最多 400 点（均匀取点） |
| 位置容差 | 1e-4 rad；位置限值放宽 1.05 倍（与 standalone 的 validate_trajectory(..., 1.05, 1.0, 1.0) 一致） |
| 扭矩限值 | 0.9×max_effort（与 standalone 的 TORQUE_VALIDATE_RATIO 一致） |

- **standalone**：`generate_excitation_trajectory` 内部调用 `validate_trajectory(refPos, refVel, refAcc, config, 1.05, 1.0, 1.0)`。
- **TLBO_E1**：优化结束后调用 `validate_trajectory_E1(refPos, refVel, refAcc, q_max, q_min, dq_max, ddq_max, robot_limb, max_effort, 1.05, 1.0, 1.0)`，逻辑与 standalone 一致。

## 4. 条件数计算方案（可选其一，两方案须一致）

当前代码里出现的条件数计算方式如下，**两方案应选同一种**便于对比。

| 方案 | 做法 | 特点 |
|------|------|------|
| **A. raw cond(H)** | H = ReMatrix 堆叠，c = cond(H) | 实现简单；H 病态或奇异时易得 Inf，优化难收敛。 |
| **B. 列归一化 + rref + κ(W_min)**（推荐，与 C++/baseGraIden 一致） | Y = ReMatrix 堆叠 → 列归一化 → rref 取线性无关列 W_min → c = sqrt(cond(W_min'*W_min)) | 数值稳定，避免 WᵀW 把条件数平方放大；点数>1500 时先亚采样到 1500。 |
| **C. cond(W_min'*W_min)**（不取 sqrt） | 同 B，但 c = cond(W_min'*W_min) | 数值为方案 B 的平方，与 diagnose_regressor_condition 一致。 |

当前两方案均已改为 **方案 B**（“之前那种”）。

## 5. 唯一差异

- **优化算法**：standalone 使用 fmincon（多轮随机初值 + 约束优化）；TLBO_E1 使用 TLBO（教师相 + 学员相，Tradeoff_Modify 保证限位）。
- **轨迹参数化**：standalone 为傅里叶阶数 order（dim×(2×order+1) 个系数）；TLBO_E1 为每关节 11 个系数（4 次谐波 + q_ini/dq_ini/ddq_ini）。因此系数维数与优化变量不同，但约束与校核在同一套限位与校核规则下可比。
