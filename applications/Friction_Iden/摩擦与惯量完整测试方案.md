## 1. 目标与原理概述

- **目标**: 对单个关节/电机，在同一套测试数据下，一次性得到  
  - 静摩擦 \(\tau_s\)  
  - 动摩擦模型参数（tanh / Stribeck / 库伦 + 粘滞）  
  - 转动惯量 \(I_a\)  
  并在同一数据上验证整体模型 \(\tau_{\text{model}} = I_a \ddot{q} + \tau_f(\dot{q})\) 的误差。
- **基本方程**:  
  - 实测力矩: \(\tau_{\text{real}} = K_\tau i\) 或直接由扭矩传感器给出  
  - 摩擦模型: \(\tau_f = f(\dot{q}; \text{摩擦参数})\)  
  - 净力矩: \(\tau_{\text{net}} = \tau_{\text{real}} - \tau_f \approx I_a \ddot{q}\)

---

## 2. 采集方案总览（Python 上位机 3 个 CSV）

同一电机需要三类轨迹，各采一条 CSV（时间统一为秒，速度为 rad/s）:

1. **电流扫描 CSV（静摩擦）**  
   - 关节位置保持不动，电流阶跃往返扫描。  
   - 从“明显刚要动之前”的最大静止电流估计  
     \[
       \tau_s \approx K_\tau \max\lvert i_{\text{still}}\rvert
     \]

2. **速度扫描 CSV（动摩擦）**  
   - 关节做缓慢加减速的锯齿/梯形速度轨迹，覆盖工作速度区间。  
   - 对每个近似匀速片段，提取 \(\dot{q}\) 与稳态电流，拟合 \(\tau_f(\dot{q})\)。

3. **加速度扫描 CSV（惯量）——分段正弦方案**  
   - 位置参考为分段定频正弦（如 10 段 × 6 s, 0.1–2 Hz, 幅值 0.5 rad）。  
   - 控制保证良好跟踪，记录 \(q, \dot{q}, i\)，用于之后的 \(\ddot{q}\) 与 \(I_a\) 拟合。

---

## 3. 轨迹与控制参数（推荐默认）

- **分段正弦轨迹**（加速度扫描）  
  - 段数: `n_segments = 10`  
  - 频率: 从 `f_start = 0.1 Hz` 到 `f_end = 2.0 Hz` 线性递增  
  - 总时长: `T = 60 s` → 每段 6 s  
  - 参考轨迹:  
    - 位置: \(q_{\text{ref}}(\tau) = A \sin(2\pi f_i \tau + \varphi_i)\)  
    - 速度: \(\dot{q}_{\text{ref}}(\tau) = A 2\pi f_i \cos(2\pi f_i \tau + \varphi_i)\)  
    - 幅值: `A = 0.5 rad`  
  - 相位连续: 每段起点 \((q,\dot{q})\) 与上一段终点一致，保证位置、速度连续。
- **限幅**  
  - 位置: \(\lvert q\rvert \le 12.5\) rad  
  - 速度: \(\lvert \dot{q}\rvert \le 30\) rad/s
- **控制增益**（仅影响跟踪，不改变参考轨迹）  
  - PD: `Kp = 20`, `Kd = 2`

## 3.1 适配所有电机的通用方案（一套参数全机型）

目标：**不按电机逐个调参**，用同一套轨迹与辨识参数，在“允许范围内”的任意电机上都能跑通并得到可比较结果。

**思路**：  
- 轨迹用**统一、保守**的物理限幅（所有电机都满足的公共上限）；  
- 辨识用**偏宽松、稳健**的阈值，避免小惯量/低 SNR 电机直接“无有效段”；  
- 仅需为每台电机配置 **K_tau** 和**数据路径**，其余保持不变。

### 通用轨迹参数（上位机 / 轨迹生成）

| 参数 | 取值 | 说明 |
|------|------|------|
| 幅值 A | **0.5 rad** | 统一值；若某关节机械范围更小，在上位机用 min(0.5, 0.3×关节行程) 等按比例缩小 |
| 频率 | f_start=**0.1** Hz, f_end=**2.0** Hz | 通用带宽，大电机跟踪更好、小电机也能跟 |
| 段数 / 时长 | **10 段 × 6 s，总长 60 s** | 统一，便于脚本里 SegmentDuration=6, NumSegments=10 固定 |
| 位置限幅 | \(\lvert q\rvert \le 12.5\) rad | 公共上限；小行程关节在上位机用更小限幅 |
| 速度限幅 | \(\lvert \dot{q}\rvert \le 30\) rad/s | 公共上限；小电机用更小速度上限 |
| 控制 | Kp=20, Kd=2 | 可统一；若某电机明显欠阻尼/过阻尼再单独微调 |

### 通用辨识参数（MATLAB 脚本内，不改电机）

在 `run_full_iden_three_files.m` 与分段正弦惯量调用里**固定**为：

| 项目 | 取值 | 说明 |
|------|------|------|
| VarianceSpeedPct | **5** | 匀速段判定，通用 |
| R2Min | **0.5** | 段有效阈值，放宽以保证小电机也有有效段 |
| AccelMeanMin | **0** | 不按 mean(\|q̈\|) 再筛，避免大电机没事、小电机全被滤掉 |
| MiddleFrac | **0.6** | 每段取中间 60% |
| LowpassCutoffHz | **12** | 速度低通，兼顾噪声与相位 |
| JMethod | **weighted_mean** | 按 R² 加权汇总，适应各段质量不一 |
| SegmentDuration / NumSegments | **6 s / 10** | 与 60 s、10 段轨迹一致 |

**每台电机只需改**：数据目录 `dd`、三个 CSV 路径、**K_tau**、摩擦模型名（若统一用 tanh 则不必改）。  

按上述“通用方案”跑出的 I_a、τ_f 参数在不同电机间**可直接对比**；若某台电机 R² 或整体 RMSE 明显偏差，再针对该台单独微调（如降 R2Min 或调 A）。

## 4. MATLAB 端整体流程（操作步骤）

1. **准备数据与参数**
   - 在 `run_full_iden_three_files.m` 顶部配置：
     - 数据目录 `dd` 与三条 CSV: `file_static`, `file_velocity`, `file_accel`
     - 力矩系数 `K_tau`
     - 摩擦模型 `FrictionModel`（如 `'tanh_viscous'` 或 `'stribeck_viscous'`）
     - 采样频率 `Fs`（与上位机一致）

2. **运行总流程脚本**
   - 在 MATLAB 中切到 `applications/Friction_Iden` 目录，执行:
     ```matlab
     run_full_iden_three_files
     ```
   - 脚本会依次完成:
     1. 三个 CSV 的速度/加速度预览  
     2. 静摩擦 \(\tau_s\) 辨识（电流扫描）  
     3. 摩擦参数辨识（速度扫描）  
     4. 惯量辨识 \(I_a\)（加速度扫描 + 摩擦补偿）  
     5. 整体模型验证与误差分析  
     6. 生成 CSV、Markdown 报告和 Excel。

3. **查看结果与图像**
   - 在 `build/` 目录下获取:
     - `friction_iden_results.csv`：每次运行追加一行（同电机+同摩擦模型只保留最新）。  
     - `全部电机参数汇总.xlsx`：所有电机最新参数汇总。  
     - `*_静摩擦_动摩擦_转动惯量.xlsx`：单电机三张表（静摩擦 / 动摩擦 / 惯量）。  
     - 若安装了 Python openpyxl/Pillow，则图会自动嵌入 Excel；同时 PNG 也保存在 `build/`。

## 5. 关键算法步骤与数据处理

### 5.1 静摩擦 \(\tau_s\)（电流扫描）

1. 从电流扫描 CSV 中读取时间、实际速度、相电流。  
2. 找到满足 “速度近似为 0” 的静止区间。  
3. 在静止区间内取最大绝对电流 \(\max\lvert i\rvert\)，换算成力矩：
   \[
     \tau_s = K_\tau \max\lvert i\rvert
   \]
4. 结果写入静摩擦表和后续摩擦模型中（可视为 Stribeck 的 \(\tau_s\) 初值）。

### 5.2 动摩擦模型（速度扫描）

1. 从速度扫描 CSV 读取时间、速度 \(\dot{q}\)、相电流 \(i\)。  
2. 通过滑动窗口与 `VarianceSpeedPct` 判定匀速段：  
   - 满足 \(\text{std}(\dot{q}) \le \text{VarianceSpeedPct}\% \times \lvert \dot{q}\rvert\) 的窗口视为匀速。  
3. 在每个匀速段内取平均速度 \(\bar{\dot{q}}\) 与平均电流 \(\bar{i}\)，换算平均力矩 \(\bar{\tau} = K_\tau \bar{i}\)。  
4. 以 \((\bar{\dot{q}}, \bar{\tau})\) 点集拟合摩擦模型参数：
   - **tanh_viscous**: \(\tau_f = \mu_s \tanh(\dot{q}/v_{\text{act}}) + \mu_d \dot{q}\)  
   - **stribeck_viscous**: \(\tau_f = (\tau_c + (\tau_s-\tau_c)e^{-(|\dot{q}|/v_s)^2})\operatorname{sign}(\dot{q}) + b\dot{q} + \tau_0\)  
   - **coulomb_viscous**: \(\tau_f = \tau_c \operatorname{sign}(\dot{q}) + b\dot{q}\)  
5. 使用最小二乘求解参数，并计算摩擦拟合 **R²** 与 **RMSE**。

### 5.3 惯量 \(I_a\)（分段正弦加速度扫描）

1. 从加速度扫描 CSV 读取时间 \(t\)、速度 \(\dot{q}\)、电流/力矩。  
2. 低通滤波速度（如 12 Hz），用数值微分得到加速度 \(\ddot{q}\)。  
3. 用摩擦模型计算每一时刻的摩擦力矩 \(\tau_f(\dot{q})\)，得到净力矩:
   \[
     \tau_{\text{net}} = \tau_{\text{real}} - \tau_f
   \]
4. 按时间将 60 s 轨迹切成 `NumSegments` 段（默认 10 段 × 6 s），每段取中间 `MiddleFrac`（默认 60%）的样本。  
5. 对每一段，构造线性模型：
   - 带偏置: \(\tau_{\text{net}} \approx J_i \ddot{q} + c_i\)（线性最小二乘，只取 \(J_i\)）  
6. 对各段计算:
   - 段内 \(J_i\)、R²、RMSE，以及 mean\(|\ddot{q}|\)。  
7. 筛选**有效段**（R² > R2Min，且可选 mean\(|\ddot{q}|\) 大于阈值），按设定策略汇总为最终惯量:
   - 加权平均: \(I_a = \sum J_i w_i / \sum w_i\)，权重 \(w_i \propto R²_i\)  
   - 或中位数: \(I_a = \text{median}(J_i)\)

### 5.4 整体拟合与误差分析

1. 在与惯量辨识**同一数据文件**（分段正弦时为加速度 CSV）上计算:
   - 模型力矩: \(\tau_{\text{model}} = I_a \ddot{q} + \tau_f(\dot{q})\)  
   - 残差: \(\varepsilon = \tau_{\text{real}} - \tau_{\text{model}}\)  
2. 剔除 \(|\ddot{q}|\) 或 \(|\tau_{\text{real}}|\) 过大的异常点后，统计:
   - 整体拟合 **RMSE** 与 **R²**  
   - 误差随时间、随速度、随加速度的分布图，用于直观评估模型是否偏大/偏小或只在大加速度时有系统偏差。
