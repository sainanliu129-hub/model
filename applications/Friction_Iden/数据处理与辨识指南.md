# 关节摩擦与惯量辨识 - 数据处理与辨识指南

## 概述

本文档说明如何使用实测数据进行摩擦和惯量参数辨识。

**数据格式**：本目录仅支持 **Python 上位机** 导出的 6/7 列 CSV（见 `Python上位机电机扫描说明.md`）。电流扫描、速度扫描、加速度扫描各一个文件。

**使用的文件**：
- `run_full_iden_three_files.m` - 三文件一次得到 τ_s、τ_c、b、I_a（推荐）
- `identify_friction_from_trajectory.m` - 摩擦辨识（被上述脚本调用）
- `identify_inertia_from_data.m` - 惯量辨识（被上述脚本调用）

---

## 1. 快速开始（使用实测数据文件）

### 1.1 三文件一次完成（推荐）

```matlab
% 1. 打开 run_full_iden_three_files.m
% 2. 修改三个 CSV 路径和 K_tau：
%    file_static   = 电流扫描 CSV（静摩擦）
%    file_velocity = 速度扫描 CSV（τ_c、b）
%    file_accel    = 加速度扫描 CSV（I_a）
%    K_tau = 2.6;  % 力矩系数

% 3. 运行
run_full_iden_three_files
```

**脚本会自动完成**：静摩擦 τ_s、库伦/粘滞 τ_c 与 b、转动惯量 I_a，并打印汇总结果。

### 1.2 方法B：分步执行

```matlab
%% 步骤1：摩擦辨识
Result_friction = identify_friction_from_trajectory(...
    'data_file', 'PD-M1-v0_v_l_leg_hip_yaw_joint_20260204-071225.csv', ...
    'K_tau', 0.5, ...                    % 力矩系数（根据实际设置）
    'stable_start', 100, ...             % 稳定段起始偏移
    'stable_end', 50, ...                % 稳定段结束偏移
    'EstimateTauS', true, ...            % 是否估计静摩擦
    'DoPlot', true);                     % 是否绘制图形

%% 步骤2：惯量辨识
Result_inertia = identify_inertia_from_data(...
    'PD-M1-acc_v0_l_leg_hip_yaw_joint_20260204-070554.csv', ...
    0.5, ...                              % K_tau
    Result_friction.tau_c, ...           % tau_c（已辨识）
    Result_friction.b, ...                % b（已辨识）
    'tau_is_current', false, ...         % joint_torque 是力矩（false）还是电流（true）
    'DoPlot', true);

%% 步骤3：查看结果
fprintf('摩擦参数: τ_c=%.4f, b=%.4f, R²=%.4f\n', ...
    Result_friction.tau_c, Result_friction.b, Result_friction.R2);
fprintf('惯量参数: I_a=%.6f, R²=%.4f\n', ...
    Result_inertia.I_a, Result_inertia.I_a_R2);
```

## 2. 数据预处理

### 2.1 读取数据文件

```matlab
% 读取数据文件（9列格式）
data = readmatrix('your_data_file.csv');

% 数据格式：time, cmd_pos, joint_pos, joint_vel, joint_torque, freq_start, freq_end, duration, amplitude_ratio
% 提取所需列
t_data = data(:, 1);           % 时间
q_cmd = data(:, 2);            % 指令位置
q_act = data(:, 3);            % 实际位置
qd_act = data(:, 4);           % 实际速度 joint_vel
tau_data = data(:, 5);         % 力矩/电流 joint_torque
freq_start = data(1, 6);      % 起始频率（扫频参数）
freq_end = data(1, 7);        % 结束频率（扫频参数）
duration = data(1, 8);         % 持续时间（扫频参数）
amplitude_ratio = data(1, 9);  % 振幅比例（扫频参数）

fprintf('数据大小: %d 行 × %d 列\n', size(data, 1), size(data, 2));
fprintf('速度范围: %.3f ~ %.3f rad/s\n', min(qd_act), max(qd_act));
fprintf('力矩范围: %.3f ~ %.3f N·m\n', min(tau_data), max(tau_data));
```

### 2.2 数据质量检查

```matlab
% 基本检查
fprintf('数据大小: %d 行 × %d 列\n', size(data, 1), size(data, 2));
fprintf('时间范围: %.2f ~ %.2f s\n', min(t_data), max(t_data));
fprintf('速度范围: %.3f ~ %.3f rad/s\n', min(qd_act), max(qd_act));
fprintf('力矩范围: %.3f ~ %.3f N·m\n', min(tau_data), max(tau_data));

% 检查采样周期
Ts = mean(diff(t_data));
fprintf('采样周期: %.4f s (%.1f Hz)\n', Ts, 1/Ts);

% 检查异常值
speed_jump = abs(diff(qd_act)) > 1.0;
if any(speed_jump)
    fprintf('警告: 发现 %d 个速度跳变点\n', sum(speed_jump));
end
```

### 2.3 数据格式说明

**数据格式**（速度轨迹和加速度轨迹都使用9列格式）：
- 输入格式：9列（`time, cmd_pos, joint_pos, joint_vel, joint_torque, freq_start, freq_end, duration, amplitude_ratio`）
- **摩擦辨识**：`identify_friction_from_trajectory.m` 已支持9列格式，**直接使用，无需转换**
- **惯量辨识**：`identify_inertia_from_data.m` 支持9列格式，**直接使用，无需转换**

## 3. 摩擦力参数辨识

### 3.1 调用辨识函数（直接使用9列格式）

```matlab
% 使用 identify_friction_from_trajectory.m（已支持9列格式，无需转换）
Result_friction = identify_friction_from_trajectory(...
    'data_file', 'velocity_trajectory_data.csv', ...  % 9列格式数据文件（直接使用）
    'K_tau', 0.5, ...                                 % 力矩系数（根据实际设置）
    'stable_start', 100, ...                          % 稳定段起始偏移
    'stable_end', 50, ...                             % 稳定段结束偏移
    'EstimateTauS', true, ...                         % 是否估计静摩擦
    'DoPlot', true);                                  % 是否绘制图形
```

**说明**：
- `identify_friction_from_trajectory.m` 已更新，**直接支持9列格式**
- 函数会自动识别数据格式（7列或9列）
- 9列格式时，会自动提取：`time`, `cmd_pos`, `joint_pos`, `joint_vel`, `joint_torque`
- `cmd_vel` 和 `cmd_current` 会自动设为0（速度控制时通常为0）

### 3.2 查看辨识结果

% 使用已辨识的摩擦参数

### 4.2 调用惯量辨识函数

```matlab
% 使用 identify_inertia_from_data.m（直接使用9列格式，无需转换）
Result_inertia = identify_inertia_from_data(...
    'acceleration_trajectory_data.csv', ...  % 加速度轨迹数据文件（9列格式）
    K_tau, ...                               % 力矩系数
    tau_c, ...                               % 已辨识的库伦摩擦
    b, ...                                    % 已辨识的粘滞摩擦系数
    'tau_is_current', false, ...            % joint_torque 是力矩（false）还是电流（true）
    'filter_window', 5, ...                  % 滤波窗口大小
    'accel_threshold', 0.01, ...             % 加速度阈值（rad/s²）
    'DoPlot', true);                         % 是否绘制图形
```

**注意**：
- `identify_inertia_from_data.m` 直接支持9列格式，无需转换
- 函数会自动提取第1列（time）、第4列（joint_vel）、第5列（joint_torque）
- 其他列（freq_start, freq_end等）会被忽略

## 7. 数据格式说明

### 7.1 输入数据格式（9列）

**速度轨迹和加速度轨迹都使用相同的9列格式**：
```
time, cmd_pos, joint_pos, joint_vel, joint_torque, freq_start, freq_end, duration, amplitude_ratio
```

**列说明**：
- `time`: 时间 (s)
- `cmd_pos`: 指令位置 (rad)
- `joint_pos`: 实际位置 (rad)
- `joint_vel`: 实际速度 (rad/s)
- `joint_torque`: 力矩或电流（根据 `tau_is_current` 参数判断）
- `freq_start`: 起始频率 (Hz) - 扫频参数（可选）
- `freq_end`: 结束频率 (Hz) - 扫频参数（可选）
- `duration`: 持续时间 (s) - 扫频参数（可选）
- `amplitude_ratio`: 振幅比例 - 扫频参数（可选）

### 7.2 摩擦辨识需要的数据

**需要列**：`time`, `joint_pos`, `joint_vel`, `joint_torque`

**使用方法**：
- **直接使用9列格式**：`identify_friction_from_trajectory.m` 已支持9列格式，无需转换
- 函数会自动提取所需列：`time`（第1列）、`cmd_pos`（第2列）、`joint_pos`（第3列）、`joint_vel`（第4列）、`joint_torque`（第5列）
- `cmd_vel` 和 `cmd_current` 会自动设为0（速度控制时通常为0）

**使用的函数**：`identify_friction_from_trajectory.m`

### 7.3 惯量辨识需要的数据

**需要列**：`time`, `joint_vel`, `joint_torque`

**直接使用**：9列格式数据可直接使用，函数会自动提取所需列

**使用的函数**：`identify_inertia_from_data.m`

## 8. 函数参数说明

### 8.1 identify_friction_from_trajectory

**必需参数**：无（使用默认文件）

**可选参数**：
- `'data_file'`: 数据文件路径，默认 `'velocity_trajectory_data.csv'`
- `'K_tau'`: 力矩系数 N·m/A，默认 `0.5`
- `'stable_start'`: 稳定段起始偏移，默认 `100`
- `'stable_end'`: 稳定段结束偏移，默认 `50`
- `'EstimateTauS'`: 是否估计静摩擦，默认 `true`
- `'DoPlot'`: 是否绘制图形，默认 `true`

### 8.2 identify_inertia_from_data

**必需参数**：
- `data_file`: 数据文件路径
- `K_tau`: 力矩系数 N·m/A
- `tau_c`: 已辨识的库伦摩擦 N·m
- `b`: 已辨识的粘滞摩擦系数 N·m·s/rad

**可选参数**：
- `'tau_is_current'`: `joint_torque` 是电流（true）还是力矩（false），默认 `false`
- `'filter_window'`: 滤波窗口大小，默认 `5`
- `'accel_threshold'`: 加速度阈值 rad/s²，默认 `0.01`
- `'DoPlot'`: 是否绘制图形，默认 `true`

## 9. 常见问题

### Q1: 数据格式不匹配
**问题**：`数据文件列数不足，至少需要5列`

**解决**：检查数据文件，确保至少有5列数据

### Q2: 摩擦辨识失败 - 样本数不足
**问题**：`|qd| >= MinSpeed 的样本数不足`

**解决**：
```matlab
% 调整稳定段参数
Result = identify_friction_from_trajectory(...
    'data_file', 'your_file.csv', ...
    'stable_start', 50, ...   % 减小起始偏移
    'stable_end', 25);         % 减小结束偏移
```

### Q3: 惯量辨识失败 - 有效数据点不足
**问题**：`警告：有效加速度数据点不足`

**解决**：
```matlab
% 降低加速度阈值
Result = identify_inertia_from_data(...
    'your_file.csv', K_tau, tau_c, b, ...
    'accel_threshold', 0.005);  % 降低阈值
```

### Q4: joint_torque 是电流还是力矩？
**判断方法**：
- 如果 `joint_torque` 的单位是 A（安培），设置 `'tau_is_current', true`
- 如果 `joint_torque` 的单位是 N·m（牛米），设置 `'tau_is_current', false`
