## 1. 概述

### 1.1 辨识目标
- **摩擦参数**：库伦摩擦 τ_c（N·m）、粘滞摩擦系数 b（N·m·s/rad）、静摩擦 τ_s（N·m）
- **惯量参数**：电机转子惯量 I_a（kg·m²）

### 1.2 前提条件
- ✅ 关节已从机器人上拆下，单独测试
- ✅ 力矩系数 K_τ 已标定（τ_real = K_τ × i）
- ✅ 数据采集系统可用（采样率 ≥ 100 Hz，建议 500 Hz）
- ✅ MATLAB 环境已配置（需要 Statistics and Machine Learning Toolbox）

### 1.3 文件清单与功能说明

| 文件名 | 功能 | 用途 |
|--------|------|------|
| `plan/generate_friction_trajectory.m` | 生成摩擦辨识轨迹 | 按关节生成速度/加速度 CSV（500Hz），**唯一实现** |
| `plan/generate_joint_trajectory.m` | 底层轨迹生成 | 被 generate_friction_trajectory 调用 |
| `identify_friction_from_trajectory.m` | 摩擦力辨识 | 从实测数据辨识摩擦参数 |
| `identify_inertia_from_data.m` | 惯量辨识 | 从实测数据辨识惯量参数 |
| `robot_algorithm/Friction/identify_joint_friction.m` | 核心摩擦辨识函数 | 被 `identify_friction_from_trajectory.m` 调用 |
| `example_generate_friction_trajectory.m` | 轨迹生成示例 | 演示如何调用 plan 中的 generate_friction_trajectory |
| `example_identify_friction.m` | 摩擦辨识示例 | 演示如何辨识摩擦 |
| `example_identify_inertia.m` | 惯量辨识示例 | 演示如何辨识惯量 |

```python
# Python 完整示例
import numpy as np
import time
import csv

# 读取轨迹
trajectory = np.loadtxt('velocity_trajectory_500Hz.csv', delimiter=',', skiprows=1)

# 准备数据记录
data_log = []
Ts = 0.002  # 采样周期 0.002 s (500 Hz)

# 开始测试
print('开始多档匀速测试...')
start_time = time.time()

for i, (t_ref, q_cmd, qd_cmd, qdd_cmd) in enumerate(trajectory):
    # 发送速度指令
    joint_controller.set_velocity(qd_cmd)  # 速度控制
    
    # 采集实际数据
    q_act = joint_controller.get_position()      # 实际位置
    qd_act = joint_controller.get_velocity()     # 实际速度
    i_act = joint_controller.get_current()       # 实际电流
    temp = joint_controller.get_temperature()     # 温度（可选）
    
    # 记录数据
    t_actual = time.time() - start_time
    data_log.append([
        t_actual,      # 实际时间
        q_cmd,         # 目标位置
        q_act,         # 实际位置
        qd_cmd,        # 目标速度
        qd_act,        # 实际速度
        0.0,           # 目标电流（速度控制时通常为0）
        i_act,         # 实际电流
        temp           # 温度
    ])
    
    # 等待下一个周期
    elapsed = time.time() - start_time
    sleep_time = (i + 1) * Ts - elapsed
    if sleep_time > 0:
        time.sleep(sleep_time)
    
    # 进度显示（每1000个点显示一次）
    if (i + 1) % 1000 == 0:
        print(f'进度: {i+1}/{len(trajectory)} ({100*(i+1)/len(trajectory):.1f}%)')

# 保存数据
output_file = 'velocity_trajectory_data.csv'
with open(output_file, 'w', newline='') as f:
    writer = csv.writer(f)
    writer.writerow(['time', 'cmd_pos', 'joint_pos', 'cmd_vel', 'joint_vel', 
                     'cmd_current', 'joint_current', 'temp'])
    writer.writerows(data_log)

print(f'数据已保存到: {output_file}')
print(f'总数据点数: {len(data_log)}')
```

```matlab
% 检查数据文件
function check_data_quality(data_file)
    % 读取数据
    data = readmatrix(data_file);
    
    % 基本检查
    fprintf('===== 数据质量检查 =====\n');
    fprintf('文件: %s\n', data_file);
    fprintf('数据大小: %d 行 × %d 列\n', size(data, 1), size(data, 2));
    
    % 检查缺失值
    missing_data = isnan(data);
    if any(missing_data(:))
        fprintf('警告: 发现 %d 个缺失值\n', sum(missing_data(:)));
    else
        fprintf('✓ 无缺失值\n');
    end
    
    % 检查时间序列
    t_data = data(:, 1);
    dt = diff(t_data);
    dt_mean = mean(dt);
    dt_std = std(dt);
    fprintf('采样周期: %.4f ± %.4f s\n', dt_mean, dt_std);
    fprintf('采样频率: %.1f Hz\n', 1/dt_mean);
    
    % 检查速度范围
    if size(data, 2) >= 5
        qd_data = data(:, 5);  % 实际速度
        fprintf('速度范围: %.3f ~ %.3f rad/s\n', min(qd_data), max(qd_data));
        
        % 检查速度跳变
        speed_jump = abs(diff(qd_data)) > 1.0;
        if any(speed_jump)
            fprintf('警告: 发现 %d 个速度跳变点（>1 rad/s）\n', sum(speed_jump));
        end
    end
    
    % 检查电流范围
    if size(data, 2) >= 7
        i_data = data(:, 7);  % 实际电流
        fprintf('电流范围: %.3f ~ %.3f A\n', min(i_data), max(i_data));
        
        % 检查异常电流
        current_abnormal = abs(i_data) > 10.0;
        if any(current_abnormal)
            fprintf('警告: 发现 %d 个异常电流点（>10 A）\n', sum(current_abnormal));
        end
    end
    
    fprintf('========================\n');
end

% 使用示例
check_data_quality('velocity_trajectory_data.csv');
```

### 4.2 数据预处理

#### 步骤1：提取稳定段数据（摩擦辨识）
```matlab
% 读取数据
data = readmatrix('velocity_trajectory_data.csv');

% 提取列
t_data = data(:, 1);
qd_act = data(:, 5);  % 实际速度
i_act = data(:, 7);   % 实际电流

% 识别速度变化点
speed_change_idx = [];
for i = 2:length(qd_act)
    if abs(qd_act(i) - qd_act(i-1)) > 0.005  % 速度变化阈值
        speed_change_idx = [speed_change_idx; i];
    end
end

fprintf('检测到 %d 个速度变化点\n', length(speed_change_idx));

% 提取稳定段（去掉前100个点和后50个点）
stable_start = 100;
stable_end = 50;
qd_stable = [];
i_stable = [];

speed_segments = [1; speed_change_idx; length(qd_act)+1];
for seg = 1:length(speed_segments)-1
    start_idx = speed_segments(seg);
    end_idx = speed_segments(seg+1) - 1;
    
    if end_idx - start_idx > stable_start + stable_end
        stable_idx = (start_idx + stable_start):(end_idx - stable_end);
        qd_stable = [qd_stable; qd_act(stable_idx)];
        i_stable = [i_stable; i_act(stable_idx)];
    end
end

fprintf('稳定段数据点数: %d\n', length(qd_stable));
```

## 8. 常见问题与故障排除

### 8.1 轨迹生成问题

#### Q1：`readmatrix` 或 `writematrix` 函数不存在
**错误信息**：`未定义函数或变量 'readmatrix'`

**原因**：MATLAB 版本过低（需要 R2019a 或更高版本）

**解决方法**：
```matlab
% 方法1：升级 MATLAB 到 R2019a 或更高版本

% 方法2：使用替代函数（修改代码）
% 将 readmatrix 替换为：
data = csvread('file.csv', 1, 0);  % 跳过第1行表头

% 将 writematrix 替换为：
csvwrite('file.csv', data);
```

#### Q2：生成的轨迹文件为空或格式错误
**检查步骤**：
```matlab
% 1. 检查文件是否存在
if exist('velocity_trajectory_500Hz.csv', 'file')
    % 2. 检查文件大小
    file_info = dir('velocity_trajectory_500Hz.csv');
    fprintf('文件大小: %d 字节\n', file_info.bytes);
    
    % 3. 尝试读取
    try
        data = readmatrix('velocity_trajectory_500Hz.csv');
        fprintf('文件读取成功，大小: %d × %d\n', size(data));
    catch ME
        fprintf('文件读取失败: %s\n', ME.message);
    end
else
    error('文件不存在');
end
```

### 8.2 数据采集问题

#### Q1：数据文件列数不对
**错误信息**：`数据文件列数不足，至少需要5列`

**解决方法**：
```matlab
% 检查数据文件格式
data = readmatrix('your_data.csv');
fprintf('实际列数: %d\n', size(data, 2));

% 如果列数不对，检查数据文件格式
% 确保格式为：time, cmd_pos, joint_pos, cmd_vel, joint_vel, cmd_current, joint_current, temp
```

#### Q2：采样周期不恒定
**检查方法**：
```matlab
data = readmatrix('your_data.csv');
t_data = data(:, 1);
dt = diff(t_data);
dt_mean = mean(dt);
dt_std = std(dt);

fprintf('平均采样周期: %.4f s\n', dt_mean);
fprintf('标准差: %.4f s\n', dt_std);
fprintf('变异系数: %.2f%%\n', dt_std/dt_mean*100);

if dt_std/dt_mean > 0.1
    warning('采样周期不恒定，变异系数 > 10%');
end
```

### 8.3 辨识问题

#### Q1：摩擦参数辨识失败 - 样本数不足
**错误信息**：`|qd| >= MinSpeed 的样本数不足`

**原因**：
- 稳定段数据点太少
- 速度变化点识别不准确
- 数据质量差

**解决方法**：
```matlab
% 方法1：调整稳定段参数
Result = identify_friction_from_trajectory(...
    'data_file', 'velocity_trajectory_data.csv', ...
    'stable_start', 50, ...   % 减小起始偏移
    'stable_end', 25);        % 减小结束偏移

% 方法2：检查数据质量
data = readmatrix('velocity_trajectory_data.csv');
qd_act = data(:, 5);
fprintf('速度非零数据点: %d / %d\n', sum(abs(qd_act) > 0.005), length(qd_act));

% 方法3：手动提取稳定段
% （见4.2节数据处理部分）
```

#### Q2：惯量辨识失败 - 有效数据点不足
**错误信息**：`警告：有效加速度数据点不足，无法辨识 I_a`

**原因**：
- 加速度过小的点太多
- 加速度计算不准确
- 数据噪声大

**解决方法**：
```matlab
% 方法1：降低加速度阈值
Result = identify_inertia_from_data(...
    'acceleration_trajectory_data.csv', ...
    K_tau, tau_c, b, ...
    'accel_threshold', 0.005);  % 降低阈值从 0.01 到 0.005

% 方法2：检查加速度计算
data = readmatrix('acceleration_trajectory_data.csv');
qd_act = data(:, 4);
Ts = mean(diff(data(:,1)));
qdd = diff(qd_act) / Ts;
qdd = [qdd(1); qdd];
fprintf('加速度范围: %.4f ~ %.4f rad/s²\n', min(qdd), max(qdd));
fprintf('有效数据点（>0.01）: %d / %d\n', sum(abs(qdd) > 0.01), length(qdd));

% 方法3：增加滤波窗口
Result = identify_inertia_from_data(...
    'acceleration_trajectory_data.csv', ...
    K_tau, tau_c, b, ...
    'filter_window', 10);  % 增加滤波窗口
```

#### Q3：拟合 R² 太低
**判断标准**：
- R² < 0.90：需要改进
- R² < 0.95：可接受
- R² > 0.95：良好

**解决方法**：
```matlab
% 1. 检查数据质量
% 2. 增加滤波
% 3. 检查模型是否匹配
% 4. 考虑更复杂的摩擦模型
```

### 8.4 结果验证问题

#### Q1：残差有明显趋势

```matlab
% 绘制残差图
```

| 惯量辨识 | `identify_inertia_from_data(data_file, K_tau, tau_c, b, ...)` | `data_file`, `K_tau`, `tau_c`, `b` | `tau_is_current`, `DoPlot` |

### 9.2 数据格式速查

**摩擦辨识数据格式**（7-8列）：
```
time, cmd_pos, joint_pos, cmd_vel, joint_vel, cmd_current, joint_current, temp
```

**惯量辨识数据格式**（9列）：
```
time, cmd_pos, joint_pos, joint_vel, joint_torque, freq_start, freq_end, duration, amplitude_ratio
```

### 9.3 参数设置参考

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| `Ts` | 0.002 s | 500 Hz 采样率 |
| `v_max` | 4.0 rad/s | 根据关节型号设置（M20: 4.67） |
| `a_max` | 30.0 rad/s² | 根据关节型号设置（M20: 31.1） |
| `K_tau` | 0.5 N·m/A | 根据实际标定值设置 |
| `stable_start` | 100 | 稳定段起始偏移 |
| `stable_end` | 50 | 稳定段结束偏移 |
| `filter_window` | 5 | 滤波窗口大小 |
| `accel_threshold` | 0.01 rad/s² | 加速度阈值 |
