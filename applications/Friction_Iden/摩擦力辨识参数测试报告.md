# 摩擦力与惯量辨识参数测试报告

## 1. 概述

本报告用于记录基于 `run_full_iden_three_files.m` 的摩擦与惯量辨识方案、可调参数及测试结果。辨识采用 Python 上位机三种扫描 CSV（电流扫描、速度扫描、加速度扫描），依次得到静摩擦 τ_s、摩擦模型参数（库伦/粘滞或 tanh）、转动惯量 I_a，并在速度扫描数据上评估整体拟合误差。

---

## 2. 辨识方案

### 2.1 三文件流程

| 步骤 | 数据文件           | 用途           | 输出 |
|------|--------------------|----------------|------|
| 1    | 电流扫描 CSV       | 静止段最大电流 | 静摩擦 τ_s = K_tau × max\|i\| |
| 2    | 速度扫描 CSV       | 匀速段速度–电流 | 摩擦模型参数（τ_c、b 或 μ_s、v_act、μ_d） |
| 3    | 加速度扫描 CSV     | 大加速度段净力矩–q̈ | 转动惯量 I_a（需先有摩擦补偿） |

- **摩擦模型**：`coulomb_viscous`（τ_c·sign(v)+b·v）、`tanh_viscous`（μ_s·tanh(v/v_act)+μ_d·v）、`stribeck_viscous`（Stribeck + 粘滞 + 偏置）。
- **Stribeck 模型**：τ_f = (τ_c+(τ_s−τ_c)e^{−(|v|/v_s)^2})sign(v) + b·v + τ_0；参数 τ_s、τ_c、v_s、b、τ_0，约束 τ_s ≥ τ_c，v_s 建议 0.001～5 rad/s。
- **惯量拟合**：τ_net = τ_real − τ_friction = I_a×q̈。脚本支持两种方案（`run_full_iden_three_files.m` 顶部 `inertia_method` 切换）：

  - **accel_segments（默认）**：匀加/匀减速段法。按 q̈ 符号与阈值切段，每段取 (mean(q̈), mean(τ_net)) 一点，过原点线性拟合 τ_net = I_a×q̈。实现：`identify_inertia_from_data.m`；可选仅用 \|q̈\|>accel_threshold、三步骤 R² 对比。
  - **segmented_sinusoidal**：分段正弦方案。轨迹为分段定频正弦（如 10 段×6 s，0.1～2 Hz）时，按时间每段取中间 60% 数据，段内拟合 τ_net = J_i×q̈，有效段（R²>0.5、可选 \|q̈\| 足够大）的 J_i 取**中位数**作为 I_a。实现：`identify_inertia_segmented_sinusoidal.m`；参数 SegmentDuration、NumSegments、MiddleFrac、R2Min、AccelMeanMin 等可调。

### 2.2 主要可调参数（脚本顶部）

| 参数 | 含义 | 典型取值 |
|------|------|----------|
| K_tau | 力矩系数 N·m/A | 由电机规格 |
| FrictionModel | 摩擦模型 | 'tanh_viscous' / 'coulomb_viscous' |
| stable_start, stable_end | 稳定段去掉首尾点数 | 100, 50 |
| VarianceSpeedPct | 匀速段判定：速度标准差≤速度的此百分比 | 5～15 |
| inertia_method | 惯量辨识方案 | 'accel_segments' / 'segmented_sinusoidal' |
| accel_threshold | 惯量拟合仅用 \|q̈\|>此值的点（rad/s²），0=全部点（仅 accel_segments） | 30～80 |
| accel_exclude_above | 剔除 \|q̈\|>此值的点，不参与拟合与整体误差；Inf=不剔除 | 500 或 Inf |
| Fs | 采样频率 Hz | 200 |

### 2.3 输出

- **build/**：飞书用 Markdown 报告、结果 CSV、保存的图（速度拟合图、加速度拟合图、整体拟合图、整体拟合误差图、整体误差图）。
- **整体拟合误差**：在速度扫描数据上计算 τ_act − (I_a×q̈+τ_f) 的 RMSE、R²（已剔除 \|q̈\|、|τ| 过大的点）。

### 2.4 结果表格各列含义（CSV / 飞书报告）

| 列名 | 含义 | 单位/说明 |
|------|------|-----------|
| 电机型号 | 电机或数据目录标识（如 43_10_40） | — |
| 辨识日期 | 本次辨识运行的日期时间 | — |
| K_tau_Nm/A | 力矩系数：τ = K_tau × 电流 | N·m/A |
| 摩擦模型 | 所用摩擦模型 | tanh_viscous / coulomb_viscous |
| tau_s_Nm | 静摩擦力矩：电流扫描静止段 max\|i\|×K_tau | N·m |
| mu_s_Nm | tanh 模型饱和力矩幅值：τ_f ≈ μ_s·tanh(v/v_act)+μ_d·v | N·m |
| v_act_rads | tanh 模型激活速度：速度在此量级时摩擦从静到动过渡 | rad/s |
| mu_d_Nms_rad | tanh 模型粘滞系数：与速度成正比的阻尼项 | N·m·s/rad |
| tau_c_pos_Nm | 库伦正向 / Stribeck 时同 τ_c | N·m |
| tau_c_neg_Nm | 库伦负向 / Stribeck 时 -τ_c | N·m |
| b_Nms_rad | 粘滞系数（库伦/Stribeck/tanh 的 μ_d） | N·m·s/rad |
| （Stribeck 时）v_act_rads 存 v_s、备注存 τ_0 | Stribeck 过渡速度与常值偏置 | rad/s、N·m |
| I_a_kgm2 | 电机转子转动惯量（加速度扫描拟合得到） | kg·m² |
| I_a_R2 | 惯量拟合 R²：τ_net 与 I_a×q̈ 线性拟合优度 | 0～1 |
| R2_friction | 摩擦拟合 R²：速度–力矩拟合优度 | 0～1 |
| RMSE_Nm | 摩擦拟合均方根误差 | N·m |
| nUsed | 摩擦辨识所用有效样本数（匀速段提取后的点数） | — |
| 备注 | 预留备注列 | — |

*飞书 Markdown 报告中还会多两行：整体拟合 RMSE、整体拟合 R²（在速度扫描上评估 τ_act 与 I_a×q̈+τ_f 的误差）。*
