# 单关节摩擦力辨识轨迹生成方案

## 一、方案概述

本方案用于生成单关节摩擦力辨识测试轨迹，支持500Hz采样频率，生成两个独立的CSV文件：
1. **速度轨迹文件**：多档匀速轨迹（用于摩擦辨识）
2. **加速度轨迹文件**：多档恒定加速度轨迹（用于I_a辨识）

### 核心特点

- ✅ 每个速度/加速度档从位置0开始
- ✅ 位置平滑过渡（无跳变）
- ✅ 时间连续
- ✅ 包含正反转测试
- ✅ 采样频率：500 Hz（采样周期：0.002 s）

## 二、速度轨迹方案

### 2.1 速度档位设置

#### 超低速档（决定静摩擦 & 库伦摩擦）
- **速度范围**：±0.01 ～ ±0.10 rad/s
- **步长**：0.01 rad/s
- **顺序**：正负交替
  - +0.01, -0.01, +0.02, -0.02, +0.03, -0.03, ..., +0.10, -0.10
- **总档数**：20档（10个正速度 × 2）

#### 低速档（库伦摩擦主贡献区）
- **速度范围**：±0.15 ～ ±0.30 rad/s
- **步长**：0.05 rad/s
- **顺序**：正负交替
  - +0.15, -0.15, +0.20, -0.20, +0.25, -0.25, +0.30, -0.30
- **总档数**：8档（4个正速度 × 2）

#### 零速档
- **速度**：0 rad/s
- **总档数**：1档

#### 中速档（粘滞摩擦主贡献区）
- **速度范围**：±0.4 ～ ±0.9 rad/s
- **步长**：0.1 rad/s
- **顺序**：先所有正速度，再所有负速度
  - +0.4, +0.5, +0.6, +0.7, +0.8, +0.9, -0.4, -0.5, -0.6, -0.7, -0.8, -0.9
- **总档数**：12档（6个正速度 × 2）

#### 高速档（验证线性假设）
- **速度范围**：±1.0 ～ ±v_max rad/s
- **档数**：10档等分
- **顺序**：先所有正速度，再所有负速度
  - +1.0, +1.1, ..., +v_max, -1.0, -1.1, ..., -v_max
- **总档数**：20档（10个正速度 × 2）

### 2.2 持续时间设置（根据SOP和参考方案）

| 速度档类型 | 速度 (|v|) (rad/s) | 运行时间 | 转动角度（参考） | 说明 |
|-----------|----------------|---------|----------------|------|
| 超低速 | 0.01 | 30 s | ~0.3 圈 | 速度小，噪声大，增加时间以获得更多数据 |
| 超低速 | 0.02 | 30 s | ~0.6 圈 | |
| 超低速 | 0.05 | 25 s | ~1.25 圈 | |
| 超低速 | 0.08 | 25 s | ~2.0 圈 | |
| 超低速 | 0.10 | 20 s | ~3.2 圈 | |
| 低速 | 0.15 | 10 s | ~2.4 圈 | |
| 低速 | 0.20 | 10 s | ~3.2 圈 | |
| 低速 | 0.25 | 8 s | ~3.2 圈 | |
| 低速 | 0.30 | 8 s | ~3.8 圈 | |
| 零速 | 0 | 10 s | - | 5000个控制周期（500Hz），增加静摩擦数据 |
| 中速 | 0.4-0.9 | 转3圈 + 1 s | ~3圈 | 增加圈数以获得更多稳定数据 |
| 高速 | 1.0-v_max | 转5圈 + 2 s，或≥6 s | ≥5圈 | 大幅增加以获得更多数据 |

**计算公式**：
- 中速档：`duration = 3 × 2π / |v| + 1.0` 秒（转3圈+1秒）
- 高速档：`duration = max(5 × 2π / |v| + 2.0, 6.0)` 秒（转5圈+2秒，或至少6秒）

**数据量说明**（500Hz采样）：
- 超低速档：每档约10,000-15,000个数据点（增加后）
- 低速档：每档约4,000-5,000个数据点（保持原设置）
- 零速档：5,000个数据点（增加后）
- 中速档：每档约10,000-15,000个数据点（转3圈，增加后）
- 高速档：每档约15,000-30,000个数据点（转5圈，增加后）

### 2.3 位置平滑过渡

- **每个速度档从位置0开始**
- **速度档切换时**：
  1. 如果当前位置或速度不为0，先添加减速段
  2. 减速段将位置和速度平滑过渡到0
  3. 然后从0开始新的速度档
- **减速参数**：
  - 减速时间：最大1.0秒
  - 减速加速度：最大10.0 rad/s²

## 三、加速度轨迹方案

### 3.1 加速度档位设置

#### 小加速度档
- **加速度范围**：±0.1 ～ ±1.0 rad/s²
- **步长**：0.1 rad/s²
- **顺序**：先所有正加速度，再所有负加速度
- **总档数**：20档（10个正加速度 × 2）

#### 中加速度档
- **加速度范围**：±1.5 ～ ±5.0 rad/s²
- **步长**：0.5 rad/s²
- **顺序**：先所有正加速度，再所有负加速度
- **总档数**：16档（8个正加速度 × 2）

#### 大加速度档
- **加速度范围**：±6.0 ～ ±a_max rad/s²
- **步长**：1.0 rad/s²
- **顺序**：先所有正加速度，再所有负加速度
- **总档数**：根据a_max确定

### 3.2 持续时间设置

- **每个加速度档**：10秒（增加100%，以获得更多数据用于I_a辨识）
- **位置平滑过渡**：与速度轨迹相同，每个加速度档从位置0开始
- **数据量**：500Hz采样下，每档约5,000个数据点

## 四、输出文件格式

### 4.1 CSV文件格式

每个CSV文件包含两列：
- **time**：时间（秒），格式：四位小数（如 0.0000, 0.0020, 0.0040）
- **cmd_pos**：关节角度/位置（弧度），格式：六位小数（如 0.000000, 0.000004, 0.000008）

**注意**：CSV文件只包含时间和关节角度两列，不包含速度和加速度。

**精度说明**：
- 500Hz采样频率下，时间步长为0.002秒，使用4位小数可精确表示
- 位置使用6位小数，可提供足够的精度用于摩擦辨识

**示例**：
```csv
time,cmd_pos
0.0000,0.000000
0.0020,0.000004
0.0040,0.000008
0.0060,0.000012
...
```

### 4.2 文件命名和保存位置

- **速度轨迹文件**：`velocity_trajectory_500Hz.csv`
- **加速度轨迹文件**：`acceleration_trajectory_500Hz.csv`
- **保存位置**：默认保存在**当前工作目录**（MATLAB的`pwd`）

**指定保存路径的方法**：
```matlab
% 方法1：使用完整路径
generate_friction_trajectory(...
    'velocity_file', '/path/to/velocity_trajectory_500Hz.csv', ...
    'acceleration_file', '/path/to/acceleration_trajectory_500Hz.csv');

% 方法2：使用相对路径
generate_friction_trajectory(...
    'velocity_file', 'data/velocity_trajectory_500Hz.csv', ...
    'acceleration_file', 'data/acceleration_trajectory_500Hz.csv');
```

**查看当前工作目录**：
```matlab
pwd  % 显示当前工作目录
```

## 七、测试建议

### 7.1 数据采集要求

- **采样频率**：≥ 500 Hz（推荐）
- **必须采集的信号**：
  - 时间 (t)
  - 目标位置 (q_cmd)
  - 实际位置 (q_act)
  - 目标速度 (qd_cmd)
  - 实际速度 (qd_act)
  - 目标电流 (i_cmd)
  - 实际电流 (i_act)
  - 温度（可选）

### 7.2 数据使用规则

- **前20%时间**：不用（加速段）
- **中间60%时间**：用于拟合（稳定段）
- **后20%时间**：不用（减速段）

### 7.3 测试检查清单

- [ ] 每个速度档都有正反向
- [ ] 稳态段 ≥ 总时间的60%
- [ ] 低速段跑够时间（≥12秒）
- [ ] 无明显速度抖动
- [ ] 温度变化不超过±5°C（否则备注）

## 八、摩擦力辨识流程

### 8.1 数据采集

1. 使用 `generate_friction_trajectory()` 生成轨迹文件
2. 在真机上执行轨迹
3. 采集数据并保存为CSV文件

### 8.2 数据处理

使用 `identify_friction_from_trajectory()` 进行辨识：

```matlab
Result = identify_friction_from_trajectory(...
    'data_file', 'velocity_trajectory_data.csv', ...
    'K_tau', 0.5, ...              % 力矩系数
    'stable_start', 100, ...       % 去掉前100个点
    'stable_end', 50, ...          % 去掉后50个点
    'EstimateTauS', true, ...      % 估计静摩擦
    'DoPlot', true);               % 绘制图形
```

### 8.3 输出结果

辨识结果包含：
- **τ_c**：库伦摩擦（N·m）
- **b**：粘滞摩擦系数（N·m·s/rad）
- **τ_s**：静摩擦（N·m，如果估计）
- **RMSE**：拟合残差均方根
- **R²**：拟合优度

## 九、注意事项

### 9.1 控制模式

- **推荐**：速度环控制
- **禁止**：位置环抖动
- **禁止**：自动摩擦补偿（如有需关闭）

### 9.2 速度稳定性要求

- 速度波动 < ±5%
- 角速度连续、不抖
- 若编码器噪声大：低速档可适当增加时间

### 9.3 位置平滑过渡

- 每个速度档从位置0开始
- 速度档切换时自动添加减速段
- 位置无跳变，平滑过渡
