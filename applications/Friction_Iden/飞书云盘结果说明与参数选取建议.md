说明文档

本文件夹包括：每个关节采用每种摩擦力模型的辨识参数表（每关节每模型一个 Excel），以及一张总表（全部电机参数汇总）。以下说明总表各列含义、如何选取模型，以及每个关节表中三个 Sheet 分别包含哪些内容。


一、总表（全部电机参数汇总.xlsx）

总表每一列的含义（与下方公式中变量一致）：

电机型号：关节/电机标识。
辨识日期：该行辨识完成时间。
K_tau_Nm/A：力矩系数，单位 N·m/A。
摩擦模型：tanh_viscous 或 stribeck_viscous。
tau_s_Nm：对应公式中 τ_s，静摩擦扭矩，单位 N·m（电流扫描得到）；Stribeck 公式中静摩擦水平。
mu_s_Nm：对应公式中 μ_s，饱和力矩幅值（tanh），单位 N·m。
v_act_rads：对应公式中 v_act（tanh 特征速度）或 v_s（Stribeck 特征速度），单位 rad/s。
mu_d_Nms_rad：对应公式中 μ_d，粘滞系数（tanh），单位 N·m·s/rad。
tau_c_pos_Nm：对应公式中 τ_c，正转向库伦/动摩擦水平，单位 N·m。tanh 行填 NaN。
tau_c_neg_Nm：对应公式中 τ_c，反转向库伦/动摩擦水平，单位 N·m。tanh 行填 NaN。
b_Nms_rad：对应公式中 b，粘滞系数（Stribeck），单位 N·m·s/rad。tanh 行填 NaN。
tau0_Nm：对应公式中 τ_0，Stribeck 常值偏置，单位 N·m。tanh 行填 NaN；独立一列，不作为备注。
I_a_kgm2：转动惯量，单位 kg·m²。
I_a_R2：惯量拟合 R²；未辨识到惯量时为 NaN。
R2_friction：摩擦拟合 R²（tanh 与 stribeck 均有）。
RMSE_Nm：摩擦拟合均方根误差，单位 N·m（tanh 与 stribeck 均有）。
nUsed：摩擦辨识有效样本数（tanh 与 stribeck 均有）。指参与摩擦拟合的采样点个数，一个时间点/一个 (角速度, 力矩) 点对计为 1，不是按阶段或段数计；与惯量辨识的有效段数无关。
备注：其他说明，可为空。
说明：公式中有几个参数表格就填几列，不在该模型公式中的列填 NaN。tanh 仅填 τ_s、μ_s、v_act、μ_d；Stribeck 仅填 τ_s、τ_c、v_s、b、τ_0（即 tau_s_Nm、tau_c_pos/neg、v_act_rads、b_Nms_rad、tau0_Nm），mu_s_Nm、mu_d_Nms_rad 在 Stribeck 行填 NaN。
说明：总表中上述三项对两种摩擦模型都会输出；若 tanh 行显示为空，多为列错位或旧版脚本，请用最新 run_full_iden_three_files 与 export 重新生成。


根据总表哪几列选取哪个模型（每个关节）

对同一电机型号的两行（一行 tanh_viscous，一行 stribeck_viscous），按下列列值决定选用哪一行的模型：

· 先看列「I_a_R2」：若只有一行为正常数值（0～1）、另一行为 NaN，则选用 I_a_R2 为正常数值的那一行的模型。
· 若两行 I_a_R2 均为正常数值，则比较列「R2_friction」和列「RMSE_Nm」：优先选 R2_friction 更大且 RMSE_Nm 更小的那一行；若接近，可倾向选 stribeck_viscous 那一行。
· 若两行 I_a_R2 均为 NaN，则比较列「R2_friction」和列「RMSE_Nm」：选 R2_friction 更大、RMSE_Nm 更小的一行作为摩擦参数；惯量 I_a 用标称值或其他实验，并在备注中说明。
· 整体拟合 RMSE、整体拟合 R² 在单电机 Excel 的「转动惯量」Sheet 中，若需进一步对比两模型整体误差，可打开两个单电机表查看后决定。


摩擦力模型原理与公式（简要）

tanh_viscous：摩擦扭矩 τ_f = μ_s·tanh( q̇ / v_act ) + μ_d·q̇。μ_s 为饱和力矩幅值，v_act 为特征速度（速度在此量级时由静摩擦过渡到动摩擦），μ_d 为粘滞系数。tanh 在零速附近平滑过渡，适合描述静–动摩擦连续变化。

stribeck_viscous：摩擦扭矩 τ_f = [ τ_c + (τ_s − τ_c)·exp(−(|q̇|/v_s)²) ]·sign(q̇) + b·q̇ + τ_0。τ_s 为静摩擦水平，τ_c 为动摩擦水平，v_s 为 Stribeck 特征速度，b 为粘滞系数，τ_0 为常值偏置。在低速区摩擦随速度增加而下降（Stribeck 效应），适合描述低速爬行与静动摩擦差异明显的场合。


二、每个关节一个表（电机名_摩擦模型_静摩擦_动摩擦_转动惯量.xlsx）

每个表包含 3 个 Sheet，内容如下。


Sheet「静摩擦」

电机型号、力矩系数 K_tau、静摩擦 τ_s（电流扫描静止段）、数据文件路径、附图（若有）。用于确认静摩擦取值与静止段电流是否合理。


Sheet「动摩擦」

电机型号、摩擦模型名称、该模型参数（与上方公式一致：tanh 为 μ_s、v_act、μ_d；Stribeck 为 τ_s、τ_c、v_s、b、τ_0）、摩擦拟合 R²、摩擦拟合 RMSE、有效样本数 nUsed、附图（若有）。tanh 与 Stribeck 均包含摩擦拟合 R²、RMSE、nUsed 三行。用于查看速度–力矩拟合质量与参数值。


Sheet「转动惯量」

电机型号、转动惯量 I_a、惯量拟合 R²、整体拟合 RMSE、整体拟合 R²、附图（若有）。用于查看惯量辨识结果与整体模型误差。
